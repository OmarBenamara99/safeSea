# Real-Time Boat Tracking Mobile App

## Overview
Mobile application for real-time boat tracking and geolocation, helping improve sea safety and reduce illegal immigration.

## Tech Stack
- **Mobile Framework:** Android Studio  
- **Backend:** Spring Boot  
- **Database:** Firebase for real-time boat routes  

## Key Features
- Tracks and geolocates boats in real time  
- Displays boats on an interactive map  
- Sends emergency rescue alerts to nearby authorities  
- Reduces sailing risks and improves sea safety

## Achievements
- Contributed to a **30% improvement in sea safety and immigration control**  
- Integrated real-time tracking and alerts for immediate response

- Implementing real-time geolocation and map-based features in mobile apps  
- Coordinating mobile apps with backend services for data storage and alerts
